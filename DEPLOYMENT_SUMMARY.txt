================================================================================
📞 TROUBLESHOOTING
================================================================================

Application Not Loading:
1. Check Gunicorn: sudo systemctl status portfolio.service
2. Check Nginx: sudo systemctl status nginx
3. Check logs: sudo journalctl -u portfolio.service -n 50
4. Verify socket: ls -l /run/portfolio.sock

Static Files Not Loading:
1. Run: python manage.py collectstatic --noinput
2. Check permissions: ls -la /home/kciil/Personal-Portfolio-Website/staticfiles
3. Reload Nginx: sudo systemctl reload nginx

500 Internal Server Error:
1. Check Gunicorn logs: sudo journalctl -u portfolio.service -n 50
2. Check Django settings: DEBUG, ALLOWED_HOSTS
3. Verify database permissions

SSL Issues:
1. Check certificate: sudo certbot certificates
2. Test renewal: sudo certbot renew --dry-run
3. Check Nginx SSL config: /etc/nginx/sites-available/portfolio
